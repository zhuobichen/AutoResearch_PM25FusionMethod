# GD-IDW 方法详情

> **创建日期**: 2026-04-22
> **状态**: 验证中

---

## 方法名称

GD-IDW (Gradient-Direction Inverse Distance Weighting)

## 核心思想

利用CMAQ局部梯度方向信息，在上风方向增强站点权重，下风方向减弱权重。

## 核心公式

### 方向调整因子

```
α_i(s) = exp(β · cos(θ_∇ - θ_i))

其中：
- θ_∇：CMAQ梯度方向（从低浓度指向高浓度）
- θ_i：站点i相对于预测点s的方向
- β：方向敏感系数（β≥0，β=0时退化为标准IDW）
```

### 最终权重

```
w_i(s) = d_i(s)^p × α_i(s)
```

### 融合预测

```
GDIDW(s) = Σᵢ w_i(s)·O_i / Σᵢ w_i(s)
```

---

## 物理意义

| 参数 | 物理含义 |
|------|----------|
| ∇CMAQ | CMAQ空间梯度，指向污染累积方向 |
| α > 1 | 站点在上风方向，增强影响 |
| α < 1 | 站点在下风方向，减弱影响 |
| β = 0 | 退化为标准VNA |

## 参数配置

|------|--------|------|
| k | 30 | 近邻数量 |

| beta | 0.5 | 方向敏感系数 |

## 创新点

1. **纯确定性方法**：无统计学习，简单可解释
2. **利用CMAQ梯度**：捕捉污染物扩散方向性
3. **上风方向增强**：污染从上风方向来，上风站点更可靠
4. **可调参数少**：只需调节β一个参数

## 预期优势

| 对比对象 | 预期改进 |
|----------|----------|
| VNA | +1~3% R²（利用方向信息） |
| AdvancedRK | 计算复杂度低（无GPR训练） |
| Downscaler | 无MCMC迭代，速度快 |

---

## 局限性

1. 需要CMAQ网格数据计算梯度
2. 梯度方向基于模型预测，可能有误差
3. β参数需要调优
