### 2.2 核心创新：PolyEnsemble

**PolyEnsemble = RK-Poly + eVNA**

核心公式:
$$
P_{PolyEnsemble} = w_1 \cdot P_{RK-Poly} + w_2 \cdot P_{eVNA}
$$
其中 $w_1 = 0.90, w_2 = 0.10$

#### RK-Poly (Polynomial Residual Kriging)
1. 使用二次多项式OLS拟合: $O = a + b \cdot M + c \cdot M^2 + e$
2. 对OLS残差进行高斯过程回归建模
3. 融合预测: $P = a + b \cdot M + c \cdot M^2 + GPR(O - (a + b \cdot M + c \cdot M^2))$

**创新点**: 多项式OLS比线性OLS更准确地捕捉模型偏差的非线性特征

#### GPR核函数
```python
kernel = ConstantKernel(10.0) * RBF(length_scale=1.0) + WhiteKernel(noise_level=1.0)
```

### 4.2 复现方法表现不佳原因分析

复现方法普遍R²<0.6，主要原因：

1. **CMAQ系统性偏低**：CMAQ均值13.5 vs 观测均值55.5，差距巨大
2. **简单偏差校正不适用大偏差**：BC_Mean, BC_Scale等方法在偏差过大时失效
3. **空间插值方法受局限**：VNA-like方法仅利用空间相关性，未能有效纠正系统偏差

### 4.3 复现方法与创新方法对比

| 对比项 | 复现方法 | 创新方法 |

| 最佳R² | 0.5783 | 0.8523 |
| 典型R² | 0.3-0.5 | 0.8-0.85 |
| 核心差异 | 简单空间插值/线性校正 | GPR非参数建模+集成 |

## 五、结论

1. **PolyEnsemble** 在所有方法中表现最优（R²=0.8523, RMSE=11.04）
2. 多项式OLS校正比线性OLS提升约 +0.0025 R²
3. RK-Poly与eVNA的集成进一步提升约 +0.0004 R²
4. **R²从原始ResidualKriging的0.8273提升到0.8523，总提升达2.50%**
5. **创新方法显著优于论文复现方法**（R²提升0.27-0.35）
