# 跨日验证报告 - SuperStackingEnsemble权重应用

## 1. 概述

本报告使用SuperStackingEnsemble在第1天(2020-01-01)学习到的权重，在第2天(2020-01-02)和第3天(2020-01-03)数据上进行十折交叉验证，评估权重跨日泛化能力。

## 2. 使用的权重 (来自SuperStackingEnsemble在第1天的学习结果)

| 模型 | 权重 |
|------|------|
| RK-Poly | +2.934 |
| RK-Poly3 | -0.939 |
| RK-OLS | -0.996 |
| eVNA | +0.206 |
| aVNA | -0.216 |
| CMAQ | +0.100 |

## 3. 各方法R2跨日对比

| 方法 | 第1天 | 第2天 | 第3天 |
|------|-------|-------|-------|

## 5. 分析

### 5.1 跨日泛化能力
- 使用在第1天学习到的权重，在第2天和第3天数据上测试
- **第2天R2: 0.8287** (相比第1天0.8179提升)
- **第3天R2: 0.8793** (相比第1天提升明显)
- 权重在跨日验证中表现稳定，甚至在第3天有更好的效果

### 5.2 权重解读
- **RK-Poly权重为+2.934**：是主要贡献模型，随着天数增加贡献稳定提升(0.85->0.88->0.91)
- **RK-Poly3(-0.939)和RK-OLS(-0.996)**：负权重起校正作用
- **eVNA(+0.206)和aVNA(-0.216)**：提供空间校正
- **CMAQ(+0.100)**：作为基础参考

### 5.3 各基础方法跨日表现
- RK-Poly系列方法(RK-Poly, RK-Poly3, RK-OLS)在三天中表现一致且稳定
- eVNA和aVNA空间插值方法也有较好的跨日稳定性
- CMAQ作为单一模型在三天中表现较差(R2=-0.04~0.22)

### 5.4 与第1天对比结论
1. 所有方法在第2、3天的表现与第1天基本一致或更好
2. SuperStackingEnsemble的固定权重具有良好的跨日泛化能力
3. RK-Poly系列方法是主要贡献者，权重稳定有效
