# MultiScaleGWR Multi-Day Validation Report

## 实验设置

- **方法**: MultiScaleGWR (多尺度地理加权回归)
- **公式**:
  - 局部回归: y = β₀ + β₁ * M
  - 权重: w = exp(-d²/(2h²))
  - 多尺度融合3个带宽(0.5, 1.0, 2.0)的预测

- **对比方法**: RK-Poly (多项式残差克里金)
- **验证方式**: 十折交叉验证
