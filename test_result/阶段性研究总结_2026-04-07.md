## 一、研究目标

### 1.1 问题定义

基于监测站点数据和CMAQ化学传输模型模拟数据，融合生成高精度的PM2.5浓度空间分布图。

### 1.2 评估指标

- **R²**：决定系数，越高越好
- **MAE**：平均绝对误差，越低越好
- **RMSE**：均方根误差，越低越好

### 1.3 数据说明

- **监测数据**：2020年全年每日PM2.5浓度
- **CMAQ数据**：2020年全年每日预测网格
- **站点数量**：约1428-1528个监测站点
- **验证方法**：10折交叉验证

## 二、基准方法

### 2.1 VNA (Voronoi Neighbor Averaging)

**公式**：
$$P_{VNA} = \sum_{i=1}^{k} w_i \cdot O_i, \quad w_i = \frac{d_i^{-2}}{\sum d_i^{-2}}$$

**原理**：使用k个最近邻监测站的观测值加权平均

**实现**：
```python
nn = NNA(method='nearest', k=15, power=-2)
nn.fit(coords_train, y_train)
vna_pred = nn.predict(coords_test)
```

### 2.2 aVNA (Additive VNA)

**公式**：
$$P_{aVNA} = M + \bar{B}_{VNA}$$
其中 $B = O - M$ 是CMAQ偏差

**原理**：CMAQ值 + 邻域偏差的平均值

### 2.3 eVNA (Multiplicative VNA)

**公式**：
$$P_{eVNA} = M \times \bar{r}_{VNA}$$
其中 $r = O / M$ 是CMAQ比值

**原理**：CMAQ值 × 邻域比值的平均值

## 三、创新方法：RK-Poly

### 3.1 方法原理

**核心思想**：将问题分解为"非线性偏差校正" + "空间残差插值"

**公式**：
$$P_{RK-Poly} = \underbrace{f(M)}_{\text{多项式校正}} + \underbrace{R^*_{GPR}}_{\text{GPR空间建模}}$$

其中：
- $f(M) = a + bM + cM^2$（二次多项式捕捉非线性偏差）
- $R^*_{GPR}$ = GPR对残差的空间插值

### 3.2 实现代码

```python
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.gaussian_process import GaussianProcessRegressor
from sklearn.gaussian_process.kernels import RBF, WhiteKernel

# 1. 多项式校正
poly = PolynomialFeatures(degree=2, include_bias=False)
m_train_poly = poly.fit_transform(m_train.reshape(-1, 1))
ols = LinearRegression().fit(m_train_poly, y_train)
pred_poly = ols.predict(m_test_poly)
residual = y_train - ols.predict(m_train_poly)

# 2. GPR残差建模
kernel = RBF(length_scale=1.0) + WhiteKernel(noise_level=1.0)
gpr = GaussianProcessRegressor(kernel=kernel, alpha=0.1, normalize_y=True)
gpr.fit(coords_train, residual)
gpr_pred, _ = gpr.predict(coords_test, return_std=True)

# 3. 融合
rk_poly_pred = pred_poly + gpr_pred
```

### 3.3 创新点

1. **多项式校正**：捕捉CMAQ偏差与模型值的非线性关系
2. **GPR空间建模**：利用高斯过程回归建模残差的空间相关性
3. **分解-组合策略**：将复杂问题分解为两个子问题分别处理

## 六、关键发现

### 6.1 RK-Poly适用条件

1. **VNA基准较低时（R²<0.88）**：RK-Poly提升显著，平均+0.02~0.05
2. **重污染天气**：CMAQ偏差大，非线性校正效果明显
3. **空间异质性强时**：GPR空间建模能捕捉局部特征

### 6.2 RK-Poly失效条件

1. **VNA基准较高时（R²>0.90）**：空间相关性简单，VNA已足够
2. **清洁天气**：CMAQ与观测接近线性关系，多项式校正无用
3. **扩散条件好**：空间平滑，局部建模反而引入噪声

### 6.3 季节规律

| 季节 | 特征 | RK-Poly表现 |
|------|------|-------------|
| **冬季（1月）** | 重污染、空间异质性强 | **最佳**，87%胜率 |
| **初冬（12月）** | 波动大、部分清洁天 | 一般，67%胜率 |
| **春季（5月）** | 过渡期、污染适中 | 一般，67%胜率 |
| **夏季（7月）** | 清洁天为主、VNA基准高 | **最差**，80%胜率但提升小 |

## 七、方法的物理意义

### 7.1 多项式校正的物理动机

- CMAQ在**高污染时系统性低估更严重**
- 二次多项式可以捕捉这种**非线性偏差**
- eVNA假设 $O = M \times r$（线性比率），但实际关系往往弯曲

### 7.2 GPR残差建模的物理动机

- 残差不是白噪声，存在**空间相关性**
- GPR的RBF核可以建模这种**空间依赖性**
- 残差来源于**局地源排放、地形效应**等空间异质因素

### 7.3 为什么VNA在清洁天更好

- 清洁天PM2.5空间分布**均匀平滑**
- 邻域平均已能很好捕捉空间相关性
- 额外建模反而引入过拟合风险

## 八、方法论贡献

### 8.1 创新点总结

1. **问题分解策略**：将"CMAQ偏差校正"与"空间残差插值"分离
2. **多项式校正**：捕捉CMAQ偏差的非线性特征
3. **GPR空间建模**：利用高斯过程回归的空间插值能力
4. **固定参数设计**：GPR核参数固定，避免过拟合

### 8.2 与已有方法对比

| 方法 | 核心思想 | 适用场景 |
|------|---------|---------|
| VNA | 邻域观测值加权平均 | 空间平滑、清洁天 |
| aVNA | CMAQ + 邻域偏差 | 线性偏差 |
| eVNA | CMAQ × 邻域比值 | 比例偏差 |
| **RK-Poly** | **非线性校正 + GPR残差** | **复杂偏差、异质空间** |

---

## 九、局限性与未来方向

### 9.1 固定参数的局限

- **无法自适应天气条件**：需要根据VNA表现动态调整
- **部分天数不如VNA**：7月有3天、12月有5天VNA更优

### 9.2 未来改进方向

1. **自适应参数选择**：
   - 当VNA R² > 0.90时，减少GPR权重
   - 当VNA R² < 0.85时，增强多项式校正

2. **集成策略**：
   - RK-Poly与VNA加权融合
   - 根据局部数据密度自适应权重

3. **引入额外变量**：
   - 气象因素（风速、边界层高度）
   - 土地利用类型
   - 人口密度

## 十、结论

### 10.1 主要结论

1. **RK-Poly是有效的创新方法**：
   - 全年60天测试，平均提升+0.011
   - 75%的天数优于VNA
   - 最大单日提升达+0.056

2. **效果与天气条件高度相关**：
   - 重污染天（冬季）：RK-Poly显著优于VNA
   - 清洁天（夏季）：VNA往往足够好

3. **物理意义清晰**：
   - 多项式校正：捕捉CMAQ非线性偏差
   - GPR空间建模：捕捉残差空间相关性

### 10.2 实用建议

| 场景 | 推荐方法 |
|------|---------|
| 冬季重污染天 | RK-Poly |
| 夏季清洁天 | VNA |
| 不确定天气 | RK-Poly（75%更优） |
| 需要稳定表现 | VNA（表现一致） |
