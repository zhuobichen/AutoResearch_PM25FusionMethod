# 北京STK时空克里金方法复现指令

## 方法名称
STK (Spatiotemporal Kriging) - 时空克里金偏差校正

## 论文来源
基于时空克里金模型估算北京地区日均PM2.5暴露量

## 方法原理

### 核心思想
时空克里金(STK)是一种考虑时空相关性的插值方法，假设PM2.5浓度在空间和时间上存在连续变化和自相关性。

### 数学模型
```
观测值: Z(s_i, t_j) = M(s_i, t_j) + R(s_i, t_j)
其中:
  - M(s_i, t_j) 是CMAQ模型值
  - R(s_i, t_j) 是时空残差，通过时空克里金插值

预测: P_STK(s_0, t_0) = M(s_0, t_0) + R_STK(s_0, t_0)
```

### 时空残差建模
1. 计算站点残差: R(s_i, t_j) = O(s_i, t_j) - M(s_i, t_j)
2. 估计时空变异函数:
   - 空间变异函数: γ_s(h)
   - 时间变异函数: γ_t(τ)
   - 时空联合变异函数: γ_st(h, τ)
3. 使用加权最小二乘法拟合理论变异函数模型

### 变异函数模型
采用分离型时空变异函数:
```
γ_st(h, τ) = γ_s(h) + γ_t(τ)
其中:
  γ_s(h) = c0 + c1 * Spherical(h)  (空间球状模型)
  γ_t(τ) = d0 + d1 * Exponential(τ) (时间指数模型)
```

### 克里金预测
对于新位置(s_0, t_0)，使用简单克里金:
```
R_STK(s_0, t_0) = Σ λ_i * R(s_i)
其中权重 λ_i 通过最小化预测方差获得
```

## 实现参数
- variogram_model: 'spherical' (空间) + 'exponential' (时间)
- spatial_range: 从数据自动估计 (数据空间范围的中位数)
- temporal_range: 从数据自动估计
- k_neighbors: 30 (用于加速计算的近邻数量)
- temporal_window: 考虑前后若干天的数据

## 输入数据
- X_obs: 监测站点坐标 (n_obs, 2) - [lon, lat]
- y_obs: 监测站点观测值 (n_obs,)
- y_model_obs: 监测站点对应的CMAQ模型值 (n_obs,)
- time_obs: 监测站点对应的时间 (n_obs,) - 天数
- X_grid: 预测网格点坐标 (n_grid, 2)
- y_model_grid: 网格点CMAQ模型值 (n_grid,)

## 输出
- y_pred: 融合预测值 (n_grid,)

## 实现注意事项
1. 如果提供时间信息，使用时空克里金；否则退化为空间克里金
2. 简化实现：使用高斯核加权代替完整克里金系统求解
3. 空间距离使用经纬度计算，考虑地球曲率校正
