# 【可执行方法规范】

## 方法名称
Additive VNA - 减法偏差法 (aVNA)

## 文献来源
- 论文标题：《PM2.5 CMAQ融合方法研究背景》
- 来源：背景文档
- 关键章节：3.2.4

## 核心公式
$$
P_{aVNA}(s_0) = M(s_0) + \sum_{i \in \mathcal{N}(s_0)} w_i \cdot \left(O(s_i) - M(s_i)\right)
$$
其中 $w_i = \frac{d_i^p}{\sum_j d_j^p}$，默认 $p = -2$

## 设计原理
aVNA将CMAQ模型值作为趋势项，通过插值近邻站点的"观测-模型偏差"来校正网格点。偏差可为正（模型偏低）或负（模型偏高）。

## 参数清单
| 参数名 | 类型 | 默认值 | 说明 |
|-------|------|-------|------|
| k | int | 30 | 近邻数量 |
| power | float | -2 | 距离权重指数 |
| min_neighbors | int | 3 | 最小近邻数 |

## 数据规格

### 输入
| 数据 | 格式 | 维度 | 单位 |
|-----|------|-----|------|
| 目标点坐标 | array | (n_target, 2) | 度 |
| 监测站点坐标 | array | (n_obs, 2) | 度 |
| 监测站点观测值 | array | (n_obs,) | μg/m³ |
| CMAQ模型值（站点） | array | (n_obs,) | μg/m³ |
| CMAQ模型值（网格） | array | (n_target,) | μg/m³ |

### 输出
| 数据 | 格式 | 单位 |
|-----|------|------|
| 融合预测值 | array | μg/m³ |

## 实现步骤

1. **计算偏差场**: 对每个监测站计算 $b_i = O(s_i) - M(s_i)$
2. **近邻搜索**: 对每个目标点找到k个最近邻
3. **偏差插值**: 用IDW插值近邻偏差 $\hat{b}(s_0) = \sum w_i b_i$
4. **预测融合**: $P_{aVNA}(s_0) = M(s_0) + \hat{b}(s_0)$
5. **边界处理**: 近邻不足时退化为纯CMAQ预测
