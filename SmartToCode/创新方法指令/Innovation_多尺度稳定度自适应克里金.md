# 多尺度稳定度自适应克里金 (Multi-Scale Stability-Adaptive Kriging, MSAK)

## 1. 方法名称

- **中文**: 多尺度稳定度自适应克里金
- **英文**: Multi-Scale Stability-Adaptive Kriging (MSAK)
- **版本**: v1.0
- **发布日期**: 2026-04-09

## 2. 核心公式

### 2.1 多尺度克里金基础

$$Z_{MSAK}(\mathbf{x}) = \sum_{s=1}^{S} w_s(\mathbf{x}) \cdot Z_s(\mathbf{x})$$

其中 $S$ 为尺度层数，$w_s(\mathbf{x})$ 为基于稳定度的空间权重。

### 2.2 稳定度自适应相关长度

$$\lambda_{PBL}(PG) = \lambda_0 \cdot \left[1 + \beta \cdot \exp\left(-\frac{|PG - PG_{crit}|}{\sigma_{PG}}\right)\right]$$

- $PG$: Pasquill-Gifford 稳定度等级 (1-6)
- $PG_{crit} = 3$: 中性稳定度临界值
- $\lambda_0$: 基础相关长度 (km)
- $\beta$, $\sigma_{PG}$: 稳定度响应参数

### 2.3 各尺度预测值

$$Z_s(\mathbf{x}) = \mu_{CMAQ,s}(\mathbf{x}) + \epsilon_s(\mathbf{x})$$

其中 $\mu_{CMAQ,s}$ 为第 $s$ 尺度下的 CMAQ 趋势，$\epsilon_s$ 为残差克里金。

### 2.4 稳定度权重融合

$$Z_{final}(\mathbf{x}) = \alpha(PG) \cdot Z_{CMAQ}(\mathbf{x}) + [1 - \alpha(PG)] \cdot Z_{GPR}(\mathbf{x})$$

$$\alpha(PG) = \frac{1}{1 + \exp[-\gamma \cdot (PG - PG_{mid})]}$$

- $PG_{mid}$: 稳定度阈值参数
- $\gamma$: 平滑参数

## 4. 关键步骤

### Step 1: 稳定度分类

```
输入: 气象数据 (风速, 辐射, 云量)
处理:
  1. 计算 Pasquill-Gifford 稳定度等级 PG ∈ {1,2,3,4,5,6}
  2. A-B 级 (PG≤2): 大气不稳定 → 短相关长度
  3. C-D 级 (PG=3-4): 中性 → 中等相关长度
  4. E-F 级 (PG≥5): 稳定 → 长相关长度
输出: PG 稳定度等级
```

### Step 2: 相关长度计算

```
输入: PG 等级
处理:
  1. λ(PG) = λ₀ × [1 + β × exp(-|PG - PG_crit|/σ_PG)]
  2. 各尺度相关长度: λ_s = λ(PG) / 2^(s-1), s = 1,...,n_scales
输出: 各尺度相关长度 {λ_1, λ_2, ..., λ_S}
```

### Step 3: 多尺度克里金插值

```
输入: 监测站点数据, 各尺度相关长度
处理 (各尺度 s):
  1. 构建变异函数: γ_s(h) = γ_nugget + γ_sill × [1 - exp(-h/λ_s)]
  2. 解克里金方程组获取权重
  3. 计算尺度预测值 Z_s(x)
输出: 各尺度预测值 {Z_1, Z_2, ..., Z_S}
```

### Step 4: 稳定度自适应融合

```
输入: CMAQ数据, GPR数据, PG等级
处理:
  1. α(PG) = 1 / [1 + exp(-γ × (PG - PG_mid))]
  2. Z_RK(x) = α(PG) × Z_CMAQ(x) + [1 - α(PG)] × Z_GPR(x)
  3. Z_MSAK(x) = Σ w_s(x) × Z_s(x), w_s 正比于 1/λ_s
输出: MSAK 融合结果
```

## 5. 物理意义

### 5.1 稳定度与相关长度

- **不稳定大气 (PG≤2)**: 湍流强烈，污染物快速扩散，相关长度短 (5-15km)
- **中性大气 (PG=3-4)**: 混合层发展充分，传输稳定，相关长度中等 (15-30km)
- **稳定大气 (PG≥5)**: 夜间或逆温层，污染物累积，相关长度长 (30-50km)

### 5.2 多尺度分解物理机制

- **大尺度 (λ长)**: 捕捉区域背景传输 (CMAQ主导)
- **中尺度 (λ中)**: 捕捉城市尺度排放源影响
- **小尺度 (λ短)**: 捕捉局地湍流扩散 (监测数据主导)
