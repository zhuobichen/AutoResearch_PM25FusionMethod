# 【可执行方法规范】

## 方法名称
AI大气化学集合生成器 (EnsAI: AI-based Ensemble Generation for Atmospheric Chemistry)

## 文献来源
- 论文标题：EnsAI: An Emulator for Atmospheric Chemical Ensembles
- 作者/年份：Sitwell / 2026年
- 关键章节：Section 3 / Section 4

## 核心公式

**GEM-MACH模型算子：**
$$
c_i = M_{GM}(e_i, \theta)
$$
其中 $e_i$ 为第i个集合成员的排放扰动，$\theta$ 为其他未扰动输入。

**集合误差协方差：**
$$
B_{ens} = L \circ \frac{1}{N-1} \sum_{i=1}^{N} \delta x_i \delta x_i^T
$$

**EnsAI模型：**
$$
\delta c_i = M_{EnsAI}(\delta e_i, u, v, T)
$$
输入：排放扰动 $\delta e$，水平风场 $u, v$，表面温度 $T$。

**U-Net架构：**
$$
[ \delta e, u, v, T ] \xrightarrow{encoder} [128, 128, 128] \xrightarrow{decoder} \delta c
$$

## 参数清单
| 参数名 | 类型 | 默认值 | 说明 |
|-------|------|-------|------|
| 输入分辨率 | int | 256×256 | 降采样后网格 |
| 原始分辨率 | int | 772×642 | GEM-MACH原始 |
| 编码器通道 | list | [128, 256, 512] | 各层通道数 |
| 输出通道 | int | 1 | 浓度扰动 |
| 排放扰动标准差 | float | 50% | 月均值比例 |
| 空间相关半宽 | float | 40km | 半高宽 |

## 数据规格
| 数据 | 格式 | 维度 |
|-----|------|-----|
| 氨排放扰动 | array | (256, 256) |
| 风场u, v | array | (256, 256) |
| 温度T | array | (256, 256) |
| 浓度扰动输出 | array | (256, 256) |
| 训练数据 | GEM-MACH集合 | ~50 ensemble members |

## 实现步骤
1. **排放扰动生成**：从各向同性高斯随机场生成排放扰动
2. **GEM-MACH集合运行**：使用扰动排放运行物理模型生成浓度集合
3. **U-Net训练**：学习从(排放扰动+气象)到浓度扰动的映射
4. **推理**：仅需80ms生成一个集合成员（vs GEM-MACH数小时）
