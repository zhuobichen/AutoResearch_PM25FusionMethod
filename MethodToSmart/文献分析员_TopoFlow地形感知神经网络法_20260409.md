# 【可执行方法规范】

## 方法名称
地形感知流场学习法 (Topography-aware pollutant Flow learning, TopoFlow)

## 文献来源
- 论文标题：TopoFlow: Physics-guided Neural Networks for high-resolution air quality prediction
- 作者/年份：Kheder et al. / 2026年
- 关键章节：Section 2 / Section 3

## 核心公式

**地形感知注意力偏置：**
$$
Attention(Q, K, V, Z) = softmax\left(\frac{QK^T}{\sqrt{d_k}} + B_{topo}(Z)\right)V
$$

其中 $B_{topo}(Z)$ 是基于高程的注意力偏置矩阵，$Z$ 为地形高程。

**风导向 patch 重排序：**
$$
P_{reordered} = reorder(P, \phi_{wind})
$$
其中 $\phi_{wind}$ 是主导风向角度，patch 顺序按风向重新排列。

**多变量输入：**
$$
X \in \mathbb{R}^{H \times W \times V}, \quad V = 15
$$
包含：6种污染物浓度、气象变量(5个：u/v风速、温度、相对湿度、气压)、坐标、时间戳、地形和人口密度。

## 参数清单
| 参数名 | 类型 | 默认值 | 说明 |
|-------|------|-------|------|
| H | int | 128 | 网格高度 |
| W | int | 256 | 网格宽度 |
| V | int | 15 | 输入通道数 |
| d_k | float | - | 注意力头维度 |
| patch_size | int | - | 图像块大小 |
| forecast_horizons | list | [12, 24, 48, 96] | 预测时效(hours) |

## 数据规格
| 数据 | 格式 | 维度 |
|-----|------|-----|
| 输入场 | 2D map | (128, 256, 15) |
| CMAQ/CAQRA重分析 | netCDF | 0.25°分辨率 |
| 地形数据(ETOPO1) | GeoTIFF | 1弧分 |
| 融合网格 | array | 15km × 15km |
| 预测输出 | (PM2.5, PM10, SO2, NO2, CO, O3) | μg/m³ |

## 实现步骤
1. **数据预处理**：将多变量数据堆叠为2D地图，使用z-score标准化气象变量，地形和人口密度归一化到[0,1]
2. **风导向Patch重排序**：按主导风向将输入数据裁剪为区域扇区，扇区内patch按风向排序
3. **地形感知注意力**：在注意力计算中加入高程偏置，惩罚跨高海拔区域的注意力权重
4. **Swin Transformer处理**：通过Swin Transformer块提取时空特征
5. **多步预测**：递归生成12-96小时预报
