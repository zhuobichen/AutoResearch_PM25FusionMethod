# 【可执行方法规范】

## 方法名称
基于无分类器引导的生成式数据同化 (GenDA: Generative Data Assimilation via Classifier-Free Diffusion Guidance)

## 文献来源
- 论文标题：GenDA: Generative Data Assimilation on Complex Urban Areas via Classifier-Free Diffusion Guidance
- 作者/年份：Giral et al. / 2026年
- 关键章节：Section 2 / Section 3

## 核心公式

**观测模型：**
$$
y = H(u^*) + \eta
$$
其中 $H$ 为观测算子，$\eta$ 为观测噪声。

**后验分布：**
$$
p(u|y, G) \propto p(u|G) p(y|u)
$$

**无分类器引导（CFG）：**
$$
\tilde{D}_\theta(u, \sigma; y) = D_\theta(u, \sigma) + \gamma [D_\theta(u, \sigma; y) - D_\theta(u, \sigma)]
$$

**回火后验采样：**
$$
p_\gamma(u|y, G) \propto p(u|G) p(y|u)^\gamma
$$

其中 $\gamma$ 控制观测约束强度（类似逆温度）。

**多尺度图消息传递：**
$$
X_{o2o} \rightarrow X_{o2r} \rightarrow X_{r2r} \rightarrow X_{r2o}
$$

## 参数清单
| 参数名 | 类型 | 默认值 | 说明 |
|-------|------|-------|------|
| γ (CFG权重) | float | - | 观测约束强度 |
| n_obs | int | 300-8000 | 观测数量 |
| 图节点数 | int | ~3×10⁵ | 原始网格 |
| 降采样节点 | int | ~5.5×10⁴ | 降采样网格 |
| S | int | - | 扩散步数 |

## 数据规格
| 数据 | 格式 | 维度 |
|-----|------|-----|
| 城市几何 | mesh | unstructured |
| 风向Φ | scalar | 标量 |
| 传感器读数S | array | (n_obs,) |
| 速度场u | vector | (N, 2) |
| 训练数据 | CFD模拟 | RANS仿真 |

## 实现步骤
1. **多尺度图构建**：创建原始网格和降采样网格（含o2o/o2r/r2r/r2o四种边）
2. **无条件去噪训练**：学习几何条件流场先验 $p(u|G)$
3. **传感器条件去噪训练**：学习观测一致性约束 $p(y|u)$
4. **CFG后验采样**：在每步反向扩散中结合无条件分支和条件分支
5. **重建输出**：生成完整速度场及其不确定性估计

## 备注
- 针对复杂城市几何的网格数据同化
- 支持稀疏传感器、固定传感器云、移动轨迹等多种感知配置
- 不需要重新训练即可泛化到新几何和风向
- 可扩展到3D城市流量和空气质量监测
