# 【可执行方法规范】

## 方法名称
深度学习区域大气化学传输模型 (Zeeman Deep Learning CTM)

## 文献来源
- 论文标题：ZEEMAN: A DEEP LEARNING REGIONAL ATMOSPHERIC CHEMISTRY TRANSPORT MODEL
- 作者/年份：Pang et al. / 2025年
- 关键章节：Section 3 / Section 4

## 核心公式

**MAE损失函数：**
$$
L = \frac{1}{C \times H \times W} \sum_{i,j,k} \| f_\theta(c_{t-1,t}; c, w, e, b) - y_{t+1} \|_{1}
$$

其中 $c$ 为浓度，$w$ 为气象条件，$e$ 为排放，$b$ 为边界条件。

**边界增强：**
$$
X_{enhanced} = [X_{conc}, X_{boundary}] \quad \text{通道维度拼接}
$$

**自回归推理：**
$$
\hat{X}_{t+\Delta t} = Model(\hat{X}_{t}, X_{t-\Delta t}; \theta)
$$
$$
\hat{X}_{t+2\Delta t} = Model(\hat{X}_{t+\Delta t}, \hat{X}_{t}; \theta)
$$

## 参数清单
| 参数名 | 类型 | 默认值 | 说明 |
|-------|------|-------|------|
| C | int | 17 | 浓度通道数 |
| H | int | 40 | 纬度网格 |
| W | int | 50 | 经度网格 |
| C_channels | int | 1800 | 特征通道宽度 |
| Swin层数 | list | [2, 6, 6, 2] | 各层Transformer块数 |
| Δt | int | 1 | 时间步长(hour) |

## 数据规格
| 数据 | 格式 | 维度 |
|-----|------|-----|
| 浓度场 | netCDF | (17, 11, 40, 50) - 3D |
| 气象3D | netCDF | (5, 11, 40, 50) |
| 气象2D | netCDF | (2, 1, 40, 50) |
| 排放 | netCDF | (27, 1, 40, 50) |
| 边界条件 | netCDF | (17, 11, 2, 90) |
| LOTOS-EUROS模拟 | netCDF | 0.1° × 0.1° |

## 实现步骤
1. **边界增强**：将3D边界值拼接到原始浓度场
2. **时空立方体嵌入**：将多变量数据组合为 $C \times H \times W$ 数据立方体，通道宽设为1800
3. **Swin Transformer处理**：通过4层Swin Transformer V2块（含下采样/上采样和跳跃连接）
4. **全连接层输出**：FCL生成最终预测
5. **自回归预测**：循环使用模型输出作为下一步输入
